Nazım Eren Version History


Nazım Eren has three distinctive release codes, Stable, Long Term Support and Beta. These release codes indicate the current status of NE along with it's progression.

  • Stable versions highlight the current implementations of new features that NE can easily handle
  • Long Term Support (LTS) versions indicate major version upgrades that will be supported, and improved indefinitely. It means, regardless of the version number, every version with an LTS extension will be included and possibly improved on newer LTS versions.
  • Beta versions indicate NE's new capacities that are currently under development. As soon as the developments can be considered Stable, they will be distributed with a new version prefix.

      Added support Mongoose, JQuery & AJAX!
